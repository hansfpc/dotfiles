2. [Install OhMyZsh](https://ohmyz.sh/#install)
3. [Download a Dummy Code Editor (VSCode)](https://code.visualstudio.com/) + Shell command
4. [Install Powerlevel10k](https://github.com/romkatv/powerlevel10k#oh-my-zsh)
